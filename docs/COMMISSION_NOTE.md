# Пояснение Для Аттестационной Комиссии

Данный каталог является репозиторной версией программной части ВКР.

Что важно для проверки:

- Исходные артефакты сохранены без изменений в `artifacts/original/`.
- Полный код ноутбука экспортирован в `exports/code/main_export.py`.
- Рабочая, воспроизводимая версия кода вынесена в Python-пакет `src/composed_linear_classifiers`.
- Ошибка старого загрузчика SECOM исправлена: целевая переменная OpenML `43587` читается из `Pass/Fail`.
- Сохраненные результаты исходного ноутбука не скрывают пропуски: default-ошибки и частичность Optuna-лога вынесены в отдельные CSV и описаны в `results/README.md`.

Для быстрой технической проверки достаточно выполнить:

```bash
pip install -r requirements-minimal.txt
python scripts/smoke_test.py
python -m unittest discover -s tests
```

Для полного пересчета эксперимента требуется установить `requirements.txt` и запустить CLI-скрипты из `scripts/`.
