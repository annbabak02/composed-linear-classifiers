# Методология Эксперимента

Экспериментальная схема воспроизводит §2.4 ВКР.

1. Все задачи приводятся к бинарной классификации с инвариантом `positive = minority`.
2. Для категориальных признаков используется `LabelEncoder`; пропуски заполняются медианой; числовые признаки стандартизируются.
3. Основная схема оценки — стратифицированная 5-fold кросс-валидация с `shuffle=True` и `random_state=42`.
4. Default-режим использует параметры моделей по умолчанию.
5. Tuned-режим использует Optuna TPE, 50 испытаний и F1 по миноритарному классу как objective.
6. При сравнении default и tuned сохраняется одинаковая схема фолдов, что делает сравнение парным.

Метрики: Precision, Recall, F1, F1-macro, balanced accuracy, ROC-AUC, PR-AUC и среднее время обучения. На SECOM и Credit Card Fraud PR-AUC интерпретируется как более информативная метрика, чем ROC-AUC, из-за сильного дисбаланса классов.
